## 정의

사용자 숙련도에 맞게 스토리 제작 복잡도를 3단계로 제공하는 UX 패턴. 초보 사용자는 키워드 선택만으로, 고급 사용자는 시스템 프롬프트 직접 작성으로 스토리를 만들 수 있도록 진입 장벽을 분리한다.

## 작동 원리

**간편 모드** (초보 사용자 대상):
- 장르, 주인공 특징, 주변 인물 특징 키워드 선택
- 선택 키워드를 바탕으로 AI가 스토리라인 자동 생성 → 사용자가 선택
- 스토리라인에 필요한 추가 정보만 입력

**일반 모드** (중급 사용자 대상):
- 기본 정보, 스토리 노트, 시작 정보, 키워드북, 엔딩 정보, 등록 정보 순서로 입력
- 스토리 임시 저장 및 등록 지원

**전문가 모드** (고급 사용자 대상):
- 시스템 프롬프트 직접 입력
- AI 동작 방식과 세계관을 세밀하게 제어 가능
