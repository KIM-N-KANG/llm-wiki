## 정의

사용자 숙련도에 맞게 스토리 제작 복잡도를 단계적으로 제공하는 UX 패턴. 초보 사용자는 키워드 선택과 AI 초안 생성으로 시작하고, 더 익숙한 사용자는 기본 정보와 스토리 노트를 직접 입력하며, 고급 사용자는 전문가 모드에서 시스템 프롬프트를 상세히 작성할 수 있다.

## 작동 원리

**간편 모드** (초보 사용자 대상):
- 장르, 주인공 특징, 주변 인물 특징 키워드 선택
- 선택 키워드를 바탕으로 AI가 스토리라인 자동 생성 → 사용자가 선택
- 스토리라인에 필요한 추가 정보만 입력
- AI가 스토리 상세 정보 또는 등록 필수 항목을 생성
- 생성한 스토리로 체험 플레이를 진행한 뒤 공개 또는 비공개로 등록

**일반 모드** (중급 사용자 대상):
- 기본 정보, 스토리 노트, 등장인물, 시작, 키워드, 엔딩, 등록 정보를 입력
- 스토리 임시 저장 및 등록 지원

**전문가 모드** (고급 사용자 대상):
- 시스템 프롬프트 상세 입력
- AI 보조 생성 또는 직접 작성
- 미리보기 확인 후 임시 저장 또는 등록
