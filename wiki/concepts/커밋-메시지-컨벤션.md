## 정의

Jira 이슈 번호와 태그를 포함해 커밋과 이슈를 자동 연결하는 팀 커밋 메시지 작성 규칙.

## 작동 원리

**형식:**
```
[KNK-{Jira이슈번호}] {태그}: {커밋 제목}
```

Jira 이슈 번호를 포함하면 해당 이슈와 커밋이 자동 연결됩니다.

**태그 목록:**

| 태그 | 사용 기준 |
|------|----------|
| `Init` | 프로젝트 초기 생성 |
| `Feat` | 새로운 기능 추가 |
| `Fix` | 버그 수정 |
| `Docs` | 문서 수정 |
| `Design` | UI·스타일·디자인 수정 |
| `CICD` | 배포·CI/CD 설정 수정 |
| `Refactor` | 기능 변화 없는 코드 구조 개선 |
| `Chore` | 설정·의존성·기타 유지보수 |

## 적용 사례

```
[KNK-1] Init: 프로젝트 생성

- 프로젝트 기본 구조 생성
- 초기 의존성 설정
```

```
[KNK-2] Feat: 로그인 페이지 구현

- 로그인 화면 추가
- 소셜 로그인 버튼 연결
```
