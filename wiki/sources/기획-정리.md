## 한줄 요약

사용자가 3단계 모드로 스토리를 만들고 AI와 채팅으로 이야기를 이어가는 인터랙티브 스토리 서비스 "마냑"의 기획 정리.

## 핵심 내용

- **마냑** — "내가 쓰고 AI가 이어가는 나만의 이야기"를 표방하는 AI 인터랙티브 스토리 서비스
- **3단계 스토리 제작 모드** — 간편(키워드 선택) / 일반(정보 직접 입력) / 전문가(시스템 프롬프트 직접 작성) 모드를 제공해 초보~고급 사용자 모두 지원
- **채팅 기반 플레이** — 텍스트(상황 묘사·대사), 캐릭터 이미지, BGM을 결합한 멀티미디어 채팅 인터페이스
- **크레딧 과금 모델** — 크레딧 구매 또는 출석체크·추천인·도전과제로 무료 충전 가능
- **LLM 모델 및 토큰 사용자 선택** — 사용자가 채팅 설정에서 LLM 모델과 출력 토큰을 직접 설정

## 주목할 인사이트

사용자가 채팅 설정에서 LLM 모델과 출력 토큰 수를 직접 조정할 수 있는 기능을 명시적으로 기획했다. 이는 AI 서비스에서 "비용 vs 품질" 선택권을 사용자에게 위임하는 드문 UX 패턴이다.
