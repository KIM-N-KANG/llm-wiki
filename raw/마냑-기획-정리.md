## 1. 서비스 개요

### 1.1. 서비스명

마냑: 내가 쓰고 AI가 이어가는 나만의 이야기

### 1.2. 한 줄 소개

마냑은 사용자가 스토리와 세계관을 3단계로 간단하게 만들고, 채팅으로 AI와 함께 이야기를 이어가는 AI 인터랙티브 스토리 서비스입니다.

## 2. 문제와 기회

### 2.1. 해결하려는 문제

기존 웹툰과 웹소설은 완성된 콘텐츠를 읽는 수동적 소비 경험이 중심입니다. 사용자가 직접 이야기를 만들고 전개에 참여하려면 스토리 설계, 세계관 구성, 캐릭터 설정 같은 높은 진입 장벽을 넘어야 합니다.

### 2.2. 시장 기회

AI 캐릭터 대화와 인터랙티브 콘텐츠에 대한 관심이 커지고 있습니다. 사용자가 단순히 콘텐츠를 소비하는 것을 넘어, 직접 이야기를 만들고 AI와 상호작용하는 경험을 원하는 만큼 쉬운 제작 도구와 몰입형 채팅 경험을 결합한 서비스 기회가 있습니다.

## 3. 타겟 사용자

### 3.1. 핵심 타겟

스토리 제작과 세계관 구성에 어려움을 느끼는 AI 인터랙티브 서비스 사용자입니다.

### 3.2. 보조 타겟

AI 캐릭터 채팅, 웹소설, 웹툰, 인터랙티브 콘텐츠를 즐기는 기존 콘텐츠 소비자입니다.

## 6. MVP 개발 기능

### 6.1. 사용자 인증

#### 6.1.1. OAuth 2.0 소셜 로그인

- Google 로그인
- 로그아웃

### 6.2. 스토리 제작

#### 6.2.1. 간편 모드로 스토리 만들기

- 장르, 주인공 특징, 주변 인물 특징 키워드 선택
- 선택한 키워드를 바탕으로 AI 스토리라인 생성
- 생성된 스토리라인 중 원하는 스토리라인 선택
- 스토리라인 완성에 필요한 추가 정보 입력
- AI가 스토리 등록 필수 항목 생성
- 스토리 생성 후, 3턴 체험 플레이
- 체험 플레이 이후, 공개/비공개 등록 여부 확인
- 공개/비공개 등록

### 6.3. 스토리 탐색

#### 6.3.1. 스토리 목록 조회

- 전체 스토리 조회
- 장르별 스토리 조회

#### 6.3.2. 스토리 상세 정보 조회

- 스토리 기본 정보 조회
- 채팅 수 확인
- 선택한 스토리로 플레이 시작

### 6.4. 스토리 플레이

#### 6.4.1. 채팅 입력

- 텍스트 직접 입력
    - 상황 묘사
    - 캐릭터 대사
- AI 추천 선택지 선택

#### 6.4.2. AI 응답 출력

- 상황 묘사 출력
- 캐릭터 대사 출력
- 다음 행동 선택지 출력

### 6.5. 채팅 관리

#### 6.5.1. 내 채팅 목록 조회

- 진행 중인 채팅 목록 조회
- 채팅 삭제

#### 6.5.2. 내 채팅 선택

- 과거 채팅 이어가기

### 6.6. 고객 지원

#### 6.6.1. FAQ

- 자주 묻는 질문과 답변 확인

#### 6.6.2. 문의하기

- 서비스 문의 작성
- 오류 신고
- 개선 의견 제출
